# Visual References

This folder stores canonical Jeffrey & Byte reference boards. Jeffrey has two supported visual variants.

## Files

- `jeffrey-byte-ab-reference.jpg`: Jeffrey Original. Slightly more mature and elongated. Best for article body illustrations, README banners and technical scenes.
- `jeffrey-byte-v1-final.jpg`: Jeffrey Q. Rounder and more approachable. Best for avatars, expressions, stickers, social covers and lightweight IP assets.
- `jeffrey-two-variants-v1-generated.png`: generated two-variant board containing Jeffrey Original and Jeffrey Q together.
- `jeffrey-original-turnaround-v1.png`: clean Original turnaround reference with front, side and back views.
- `jeffrey-q-turnaround-v1.png`: clean Q turnaround reference with front, side and back views.
- `byte-expression-sheet-v1.png`: clean Byte expression and pose sheet.

## Usage Rule

Both variants are valid. Choose the variant by use case before writing prompts or reviewing output.

- Default for article illustrations: Jeffrey Original.
- Default for avatar / sticker / expression assets: Jeffrey Q.
- If the user explicitly names a variant, follow the user's choice.

Use the turnaround and expression sheets for stricter character consistency. Use the earlier boards for broader style, palette and asset-system context.
