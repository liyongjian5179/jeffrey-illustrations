# Jeffrey Example Illustrations

This folder contains Jeffrey & Byte example illustrations that correspond one-to-one with the original Ian Xiaohei example themes.

The files preserve the original example numbering and semantic themes, but the images are regenerated for the Jeffrey & Byte IP system.

## Variants

- `original/`: Jeffrey Original, for article body illustrations, README banners and technical scenes.
- `q/`: Jeffrey Q, for avatars, stickers, social covers and lighter IP assets.

## Usage Rule

Use these examples for calibration only: density, metaphor type, character participation, color and whitespace.

Do not copy a generated example as the default answer for a new user request. Rebuild the visual metaphor from the current article or topic.
