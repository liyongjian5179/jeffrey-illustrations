# Byte Character Bible

Byte 是 Jeffrey 的 AI 小助手，也是项目的第二视觉符号。

## 一句话设定

Byte 是一个圆滚滚、蓝黄配色、可靠但偶尔犯迷糊的小机器人。它陪 Jeffrey 学习、调试、写作和折腾服务器。

## 外观

- 小型机器人
- 圆润身体
- 两个简单点状眼睛
- 蓝黄奶白配色
- 身上可以有小小的 `J` 或 `>` 符号
- 不要复杂机械细节
- 不要未来战争机器人感

## 性格

- 好奇
- 活泼
- 有一点点呆
- 乐于帮助
- 喜欢递工具、举牌、搬小物件

## 常见动作

- 递咖啡
- 抱 Docker 小鲸鱼
- 搬服务器
- 举 MCP 拼图
- 递 Obsidian 笔记
- 拿 Prompt 卡片
- 在 Jeffrey 旁边看屏幕
- 部署成功时一起庆祝

## 使用原则

Byte 必须参与故事，不应只是装饰物。

如果画面中有 Byte，它最好承担一个明确动作：递、举、搬、看、修、庆祝、提醒。
