# Jeffrey Character Bible

Jeffrey 是这个项目的默认个人 IP 角色。

## 一句话设定

Jeffrey 是一个温和、专注、喜欢折腾 AI、DevOps、HomeLab、Obsidian 和个人知识管理的技术创作者。

## 外观

- 年轻男性技术人 / 运维工程师 / AI Builder
- 气质年轻、圆润、亲和，不要成熟帅气头像感
- 黑色或深蓝黑短发，不要棕发
- 无胡子、无八字胡、无络腮胡
- 圆框眼镜
- 深蓝色头戴式耳机，黄色内圈，头梁偏厚
- Jeffrey Blue Hoodie，左胸黄色 `J` 标识，版型宽松圆润
- 牛仔裤，浅蓝水洗质感
- 白色运动鞋
- 支持 Original 和 Q 两版比例，不能混成第三种随机比例
- 脸部保持简单手绘：小鼻子、简单嘴型、温和眼神，不要半写实成人五官

## 两版形象

Jeffrey 的形象需要两版并存：

- **Jeffrey Original**：比例稍成熟、身体更修长，整体更接近正文插画角色。适合中文技术文章正文配图、README 横图、技术场景插画、方法论图。
- **Jeffrey Q**：头身更圆润、更亲和，整体更适合 IP 识别和轻量传播。适合头像、表情、贴纸、社媒封面、公众号配图、Skill 图标。

两版共享：圆框眼镜、Jeffrey Hair™、深蓝耳机、黄色耳机内圈、Jeffrey Blue Hoodie、黄色 `J` 标识、牛仔裤、白色运动鞋、Byte 小助手。

## Jeffrey Hair™

最终发型不是单一现成发型，而是：

> 短碎盖为主，轻微自然偏向；要圆、短、碎，不要明显四六分

具体规则：

- 两侧自然推短，不要明显渐变
- 顶部短而圆，整体像一个紧凑的圆形发帽
- 只允许轻微自然偏向，不要清晰分线
- 前额自然短碎发，不遮眼睛
- 发量有层次但不要高耸蓬松
- 顶部略乱、小发束清晰，后脑勺圆润
- 不油头
- 不背头
- 不纹理烫
- 不长刘海
- 始终保持短发
- 不要中分、明显四六分、窗帘刘海、高耸卷翘刘海、头顶尖刺感

Prompt 固定写法：

```text
short rounded choppy hair, compact cap-like silhouette, small uneven hair clumps, soft off-center flow without a visible hard part line, rounded back head, short messy bangs above the eyebrows, not covering the eyes, no visible 4:6 part, no center part, no curtain bangs, no tall wavy quiff, no spiky crown, no pompadour, no undercut, no long fringe
```
