# Jeffrey Character Bible

Jeffrey 是这个项目的默认个人 IP 角色。

## 一句话设定

Jeffrey 是一个温和、专注、喜欢折腾 AI、DevOps、HomeLab、Obsidian 和个人知识管理的技术创作者。

## 外观

- 年轻男性技术人 / 运维工程师 / AI Builder
- 无胡子、无八字胡、无络腮胡
- 圆框眼镜
- 深蓝色头戴式耳机，黄色内圈
- Jeffrey Blue Hoodie，左胸黄色 `J` 标识
- 牛仔裤
- 白色运动鞋

## Jeffrey Hair™

最终发型不是单一现成发型，而是：

> 70% 微分碎盖 + 30% 港风四六分

具体规则：

- 两侧自然推短，不要明显渐变
- 顶部长度约 6–8cm
- 微四六分
- 前额自然碎发，不遮眼睛
- 略蓬松
- 不油头
- 不背头
- 不纹理烫
- 不长刘海
- 始终保持短发

Prompt 固定写法：

```text
short textured side-part hairstyle, soft 4:6 part, natural volume, clean side taper, slightly messy short bangs, not covering the eyes, no pompadour, no undercut, no long fringe
```

## 气质

- 温和
- 专注
- 技术宅
- 轻松折腾感
- 可靠但不商务
- 聪明但不硬核黑客
