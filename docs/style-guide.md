# Style Guide

## 风格定义

Jeffrey Warm Tech Illustration：

- 温暖手绘
- 蜡笔 / 彩铅质感
- 粗线条
- 高留白
- 蓝黄品牌色
- 轻松但不幼稚
- 技术感来自动作、物件和隐喻，不靠黑客氛围

## 构图原则

- 一张图只表达一个核心观点
- 主体占画面 40%–60%
- 背景尽量干净
- 中文标注尽量短
- 不要复杂 UI、复杂架构图或密集文字

## 必须保留

- Jeffrey Blue Hoodie
- 黄色 J Logo
- 圆框眼镜
- 蓝色耳机
- Jeffrey Hair™
- Byte（当场景需要互动时）

## 禁止

- 赛博朋克
- 黑客暗色终端墙
- 商务精英风
- 西装
- 胡子
- 儿童绘本过度幼稚
- 3D 渲染
- 复杂阴影 / 渐变 / 写实材质
