# Jeffrey Brand Guide

## Brand Positioning

Jeffrey Illustrations 是面向 AI 技术创作者的温暖手绘插画系统。

关键词：

- AI Builder
- DevOps
- PKM
- HomeLab
- Obsidian
- Warm Tech Life

## Color System

| Token | Hex | Usage |
| --- | --- | --- |
| Jeffrey Blue | `#5379E8` | Hoodie 主色 |
| Deep Blue | `#2F56C5` | 耳机、重点描边 |
| Warm Yellow | `#FFD84A` | J Logo、耳机内圈、Byte 点缀 |
| Cream | `#FFF9EE` | 默认背景 |
| Denim | `#88A8F5` | 牛仔裤 |
| Soft Gray | `#D7DCE8` | 辅助线、极少量阴影 |

## Brand DNA

看到这些元素，就应该想到 Jeffrey：

- Jeffrey Blue Hoodie
- 黄色 J Logo
- 深蓝耳机 + 黄色内圈
- 圆框眼镜
- Jeffrey Hair™
- Byte 小机器人
- 温暖蜡笔手绘质感

## Logo 使用

`J` 只用于小面积识别：

- Hoodie 左胸
- 杯子
- 电脑贴纸
- Byte 身体小标识

不要把 `Jeffrey` 大字写在衣服上。
