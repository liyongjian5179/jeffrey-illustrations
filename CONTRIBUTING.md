# Contributing

Thanks for your interest in Jeffrey Illustrations.

This repository intentionally keeps the structure lineage of `helloianneo/ian-xiaohei-illustrations`. Contributions should respect that recognizable Skill-oriented shape while extending the Jeffrey & Byte IP system.

Before contributing images or prompts, please read:

- `docs/character.md`
- `docs/style-guide.md`
- `jeffrey-illustrations/references/qa-checklist.md`

## Contribution rules

- Keep Jeffrey's character DNA consistent.
- Keep Byte simple and story-driven.
- Use short Chinese labels only when needed.
- Include the prompt used to generate or edit each illustration when possible.
- Do not submit images that look like cyberpunk, business elite, 3D render, anime, or generic stock illustration.
