# QA Checklist

Before accepting an image, check:

## Character

- [ ] Jeffrey has short Jeffrey Hair™
- [ ] No beard / mustache
- [ ] Round glasses
- [ ] Deep blue headphones with yellow inner ring
- [ ] Jeffrey Blue hoodie
- [ ] Small yellow J logo on hoodie
- [ ] Jeans and white sneakers

## Byte

- [ ] Byte is blue-yellow-cream
- [ ] Byte is simple and round
- [ ] Byte participates in the scene if present

## Style

- [ ] Warm hand-drawn crayon / colored pencil feel
- [ ] Cream or clean light background
- [ ] High whitespace
- [ ] No cyberpunk / no business elite / no 3D render

## Communication

- [ ] One image = one idea
- [ ] Jeffrey or Byte performs the core action
- [ ] Chinese labels are short and readable
- [ ] Not a PPT slide
- [ ] Not a copy of old examples
