# QA Checklist

Before accepting an image, check:

## Character

- [ ] Jeffrey has short Jeffrey Hair™
- [ ] Hair is black or deep navy-black, not brown
- [ ] Hair is compact and rounded with short choppy clumps, not tall, wavy or visibly parted
- [ ] Face is simple and cute, not semi-realistic or adult-handsome
- [ ] No beard / mustache
- [ ] Round glasses
- [ ] Deep blue headphones with yellow inner ring
- [ ] Jeffrey Blue hoodie
- [ ] Small yellow J logo on hoodie
- [ ] Jeans and white sneakers
- [ ] Uses either Original or Q proportions, not a third body type

## Byte

- [ ] Byte is blue-yellow-cream
- [ ] Byte is a compact mascot robot, not a large floating orb or drone
- [ ] Byte has simple dot eyes, blue face screen and yellow side accents
- [ ] Byte does not have a large "Byte" text label in normal scene illustrations
- [ ] Byte participates in the scene if present

## Style

- [ ] Warm hand-drawn crayon / colored pencil feel
- [ ] Cream or clean light background
- [ ] High whitespace
- [ ] No cyberpunk / no business elite / no 3D render

## Communication

- [ ] One image = one idea
- [ ] Jeffrey or Byte performs the core action
- [ ] Chinese labels are short and readable
- [ ] Not a PPT slide
- [ ] Not a character design board unless explicitly requested
- [ ] No faded alternative sketch thumbnails unless explicitly requested
- [ ] Not a copy of old examples
