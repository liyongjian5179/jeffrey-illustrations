# Examples

This folder stores low-frequency visual calibration images for the Jeffrey Illustrations skill.

Use examples to check density, character consistency, color and mood. Do not copy compositions directly.

## Character Reference

- `jeffrey-byte-ab-reference.jpg`
- `jeffrey-byte-v1-final.jpg`
- `jeffrey-original-turnaround-v1.png`
- `jeffrey-q-turnaround-v1.png`
- `byte-expression-sheet-v1.png`

`jeffrey-byte-ab-reference.jpg` stores the Jeffrey Original reference. `jeffrey-byte-v1-final.jpg` stores the Jeffrey Q reference.

Use the Jeffrey turnaround sheets for body proportion and outfit consistency. Use the Byte expression sheet for Byte's shape, markings and action library.

## Generated Example Sets

- `original/`: 14 Jeffrey Original examples corresponding to the Ian Xiaohei example themes.
- `q/`: 14 Jeffrey Q examples corresponding to the same themes.

The numbering and theme names match the original Xiaohei example lineage, but the images are regenerated for Jeffrey & Byte.
